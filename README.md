# Flask-Application-Final-Project
# Simple Flask App with Plotly.js Visualization

This is a simple Flask application that visualizes Black Publications in the United States from 1859-2025 using Leaflet.js.

# What This App Does: 
  - Reads data from the Black_Publications.csv
  - Displays Three Interactive Visualizations:
      U.S. Map of the United States with popup boxes
      Black Publications search bar
      Sildebar for Black Publications years
