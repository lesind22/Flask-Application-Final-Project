# Flask-Application-Final-Project
# Simple Flask App with Leaftlet.js Visualization

This is a simple Flask application that visualizes Black Publications in the United States from 1859-2025 using Leaflet.js.

# What This App Does: 
  - Reads data from the Black_Publications.csv
  - Displays Three Interactive Visualizations:
    1. Geographic distribution (U.S. Map of Publications from state-to-state)
    2. Detailed data (List of Publications)
    3. Direct access to sources (Digitized Link to Publications)
      

# How to Run:

	python3 -m venv venv
